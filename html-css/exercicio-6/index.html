Principais formatações

Negrito / Destaque

Modelo não semantico <b>

Nessa frase temos termos em negrito usando a tag <b>,que é uma tag desatualizada.
não deve ser ultilizada

Modelo Semantico <strong>

Nessa frase temos termos em destaque usando a tag <strong>, que é uma tag focada em conteudo, o que condiz com as regras que compõe a html

Italico

Modelo não semantico <i>

Nessa frase temos termos em italico usando a tag <i>,que é uma tag desatualizada.
não deve ser ultilizada

Modelo semantico <em>

Nessa frase temos termos em italico usando a tag <em>, que é uma tag focada em conteudo, o que condiz com as regras que compõe a html

Texto marcado

Podemos tambem criar um texto marcado usando a tag <mark>,que não é reconhecida pelo vscode. (selecione o texto que quiser marcar e pressione ctrl+shift+p e digite mark)

Teste de mudança de cor da marcação usando css

Texto grande/Texto Pequeno

Este paragrafo esta divido em texto grande e pequeno.
Ultilizando as tags <big> e <small> sendo que a tag big caiu em desuso por falta de uso semantico

Texto deletado

Podemos marcar um texto como excluido, para indicar que ele deve ser lido, mas não considerado ultilizando a tag <del>

Texto inserido

Podemos marcar um texto como inserido, para dar enfase e indicar que ele foi adicionado depois ultilizando a tag <ins>

Texto sublinhado(desuso)

É possivel tambem sublinhar um texto ultilizando a tag <u> que caiu em desuso por falta de semantica

Texto sobrescrito

Para textos a cima do texto principal da linha se ultiliza a tag <sup>

Texto subscrito

Para textos a baixo do texto principal da linha se ultiliza a tag <sub>
